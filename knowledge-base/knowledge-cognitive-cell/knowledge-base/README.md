# Knowledge Base

This folder stores reusable knowledge entries for the NSEAP AI Learning Operating System.

Use the matching folder and template for each new item.

## Folders

- `00-overview`: system-level explanations.
- `01-course`: course knowledge.
- `02-challenges`: challenge knowledge.
- `03-prompts`: reusable prompts.
- `04-faq`: common questions.
- `05-best-practices`: best practices and operating methods.
- `06-projects`: project cases and examples.
- `07-agents`: agent roles, context, and usage notes.
