# Builder FAQ

## Q: Where should I put a new knowledge item?

Choose the folder that matches the primary use case. If it supports multiple modules, choose the main category and add related links.

## Q: Do I need metadata?

Yes. Metadata makes the item easier to search, review, and use with agents later.

## Q: Can I submit draft content?

Yes. Mark `status: draft` in the frontmatter.
