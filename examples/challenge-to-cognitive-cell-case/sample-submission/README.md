# Sample Submission: C01 Build a Challenge Submission Repo

## What I Built

I created a sample GitHub repository structure for submitting an Elite20 Challenge.

## Why It Matters

This helps future students understand what a complete Challenge submission should include.

## Files

- `README.md`: explains the submission
- `submission.yaml`: structured submission metadata
- `ai-log.md`: records AI collaboration
- `reflection.md`: short AAR

## Limitation

This is a documentation-only sample and does not include runnable code.
