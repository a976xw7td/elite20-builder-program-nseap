# NSEAP Knowledge Base Intro

## Purpose

Explain the role of the Knowledge Base in the Elite20 Builder Program.

## Content

The Knowledge Base is the shared memory layer for the NSEAP AI Learning Operating System.

It stores course knowledge, challenge guidance, prompts, FAQs, best practices, project cases, and agent context in a consistent format.

## When To Use

Use this entry when introducing why the Knowledge Team exists and how it supports other teams.
