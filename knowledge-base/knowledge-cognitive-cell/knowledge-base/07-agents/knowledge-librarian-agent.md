# Knowledge Librarian Agent

## Purpose

Define a future agent role that helps users find, organize, and improve knowledge-base entries.

## Role

The Knowledge Librarian Agent helps students, teachers, and builders locate the right knowledge item and suggests where new knowledge should be added.

## Capabilities

- Search by type, audience, and tags.
- Recommend related entries.
- Check whether an entry has complete metadata.
- Suggest the right template for new content.
