# Sample AI Study Assistant Project

## Purpose

Show what a simple project case entry should include.

## Situation

Students need a lightweight assistant that helps plan study sessions and review progress.

## Ontology

Key concepts include learner, study goal, available time, difficulty, study plan, progress review, and reflection prompt.

## Workflow

```text
Collect learning goal
-> collect time and difficulty
-> generate study plan
-> ask reflection question
-> review usefulness
-> improve assistant prompt
```

## Skills

This project uses situation modeling, assistant prompt design, structured conversation design, and output evaluation.

## Solution

Create an AI assistant that asks for the learning goal, time available, and current difficulty, then generates a study plan and reflection prompt.

## Deliverables

- Assistant prompt.
- Example conversation.
- Short demo.
- Reflection notes.

## Evaluation

Evaluate whether the assistant produces a useful plan, asks relevant questions, and helps the learner reflect.
