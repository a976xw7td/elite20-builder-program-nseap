# Knowledge Capture

## FAQ Candidates

### Do all submissions need an AI log?

Yes. Elite20 evaluates not only the final artifact, but also how the Builder collaborated with AI.

### Can a documentation-only submission pass?

It can pass only if the Challenge is documentation-focused. For build-focused Challenges, a runnable or inspectable artifact is expected.

## Common Mistake

Students may include a README but forget to explain how future students can reuse the submission.

## Best Practice

A strong Challenge submission should include:

- clear README
- structured `submission.yaml`
- AI collaboration log
- reflection / AAR
- runnable or inspectable artifact
- known limitations

## Skill Improvement

The Repo Completeness Check Skill should check not only file presence, but also whether each file contains meaningful content.
