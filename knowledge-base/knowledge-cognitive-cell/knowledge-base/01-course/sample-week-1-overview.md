# Week 1 Overview

## Purpose

Give students and teachers a clear starting point for Week 1 of the Vibe Coding course.

## Learning Goal

Understand what an AI-native learning workflow looks like and complete the first small AI-assisted creation task.

## Content

Week 1 should introduce the course purpose, expected learning behavior, basic AI tool usage, and the first challenge.

## Graph Notes

This entry can become a future `CourseUnit` node. It includes the first challenge and uses a coding coach prompt as support.

## Example Activities

- Set up required accounts and tools.
- Ask an AI assistant to explain a coding concept.
- Complete a small guided creation task.
- Reflect on what AI helped with and what still required human judgment.
